머신러닝

머신러닝은 컴퓨터 시스템이 명시적인 프로그래밍 없이 경험을 통해 학습하고 개선할 수 있게 하는 인공지능의 한 분야입니다.

딥러닝

딥러닝은 여러 층의 인공 신경망을 사용하여 데이터로부터 복잡한 패턴을 학습하는 머신러닝의 하위 분야입니다.

비전딥러닝

비전딥러닝은 컴퓨터 비전 작업을 수행하기 위해 딥러닝 기술을 적용하는 분야로, 이미지 인식, 객체 검출 등에 사용됩니다.

자연어처리

자연어처리(NLP)는 컴퓨터가 인간의 언어를 이해, 해석, 생성할 수 있도록 하는 AI의 한 분야입니다.
